# 👽 XENO-GEN // Procedural ET Asset Generator

O **XENO-GEN** é um utilitário web voltado para desenvolvedores de jogos e artistas que precisam criar assets de pixel art extraterrestres de forma rápida e procedural. Com foco em estética "grotesca" e simetria orgânica quebrada, o gerador permite criar milhares de variações únicas de criaturas alienígenas prontas para serem usadas em jogos 2D.

## 🚀 Funcionalidades Principais

- **Geração por Semente (Seed):** Controle total sobre as variantes. Use a mesma semente para recriar um espécime ou gere aleatoriamente para novas surpresas.
- **Controle de "Grotesco" (Dials):** Ajuste parâmetros para definir a personalidade da criatura:
  - **Asymmetry:** Define o quão "deformada" ou assimétrica a criatura será.
  - **Noise & Mutation:** Adiciona detalhes orgânicos e deformidades únicas.
  - **Slime:** Controla a presença de detalhes viscosos nos sprites.
- **Recursos Customizáveis:** Adicione ou remova Antenas, Escoriações (Scars) e Slime Drips com um clique.
- **Animação Automática:** O motor gera automaticamente quadros de animação (*Idle, Blink, Breathe, Twitch*) para dar vida ao espécime.
- **Ferramentas de Exportação Profissional:**
  - **Sprite Sheet Generator:** Crie folhas de sprites completas com múltiplas variações baseadas em uma semente.
  - **Exportação PNG:** Salve o asset diretamente no seu computador.
  - **Batch Preview:** Visualize 6 variantes de uma vez para escolher a melhor.

## 🛠️ Tecnologias Utilizadas

- **HTML5 Canvas:** Para a renderização procedural dos pixels e animações em tempo real.
- **Vanilla JavaScript:** Lógica de algoritmos de simetria espelhada e geradores de ruído orgânico.
- **CSS3:** Interface temática de terminal "Sci-Fi" de alta tecnologia.

## 🎮 Como Usar

1. Acesse o [XENO-GEN](https://jjunninho.github.io/Alien_Generator/).
2. Escolha a resolução desejada (**16x16**, **24x24** ou **32x32**).
3. Ajuste os sliders de **Asymmetry** e **Slime** para o estilo desejado.
4. Clique em **GENERATE** ou **⚡ RANDOM** para ver novos espécimes.
5. Para desenvolvedores: Use o **BUILD SHEET** para criar uma folha de sprites e clique em **EXPORT PNG** para usar no seu projeto (como Unity, Godot ou GameMaker).
